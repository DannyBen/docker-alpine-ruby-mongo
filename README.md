Docker Alpine Ruby Mongo Tools Image
==================================================

This image is designed to be a mongo worker with ruby rupport.

Whats in the box
--------------------------------------------------

- Alpine linux
- Ruby + Bundler
- Build tools (for building native extension gems)
- Mongo Tools (mongodump, mongoexport, mongoimport etc)
